Ingeniero de Software senior

Tech Solutions Inc. | Remoto

Detalles del puesto

Tech Solutions Inc. esta buscando un Ingeniero de Software Senior altamente motivado y experimentado para unirse a nuestro equipo remoto. El candidato ideal tendra una solida formación en desarrollo de software, con experiencia en el diseño, desarrollo e implementacion de soluciones de software escalables y de alto rendimiento. Como ingeniero de Software Senior, usted sera responsable de liderar proyectos de desarrollo, mentorizar a ingenieros junior y colaborar con equipos multifuncionales para entregar productos de software de alta calidad

Responsabilidades

  • Diseñar, desarrollar y mantener aplicacione web utilizando tecnoligas modernas.
  • Colaborar con equipos de producto y diseño para definir y entregar nuevas caracteristicas
  • Escribir codigo limpio, eficiente y bien documentado
  • Realizar revisiones de codigo y proporcionar retroalimentacion constructiva a los miembros del equipo

Requisitos

  • Licenciatura en informatica o campo relacionado.
  • Minimo de 5 años de experiencia en desarrollo de software.
  • Experiencia con frameworkjs de JavaScript (por ejemplo, React, Aungular, Vue.js)
  • Familiaridad con metodologias agiles y herramientas de control de versiones (por ejemplo, Git).

Acerca de la empesa

Tech Solutions Inc. es una empresa de tecnologia innovadora que se centra en la creacion de soluciones de software de vanguardia con diversas industrias. Estamos comprometidos con el fomento de un entorno de trabajo colaborativo e inclusiv donde cada empleado pueda prosperar y crecer profesionalmente. Ofrecemos salarios competitivos, beneficios, integrales y oportunidades de desarrollo profesional continuo.
